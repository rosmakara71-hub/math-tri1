ពិន្ទុ៖ 0 / ១០០

លំហាត់ ១. គេឱ្យស្វីតនព្វន្ត $(u_n)$ មានតួទី១ $u_1 = 3$ និងផលសង្គមរួម $d = 4$។ គណនាតួទី $10$ នៃស្វីតនេះ។ (៥ ពិន្ទុ)
លំហាត់ ២. គណនាផលបូក $10$ តួដំបូងនៃស្វីតនព្វន្ត៖ $2 + 5 + 8 + \dots + u_{10}$ (៥ ពិន្ទុ)
លំហាត់ ៣. គេឱ្យស្វីតធរណីមាត្រ $(v_n)$ មានតួទី១ $v_1 = 2$ និងផលធៀបរួម $q = 3$។ គណនាតួទី $5$ នៃស្វីតនេះ។ (៥ ពិន្ទុ)
លំហាត់ ៤. គណនាផលបូក $5$ តួដំបូងនៃស្វីតធរណីមាត្រ៖ $1 + 2 + 4 + 8 + 16$ (៥ ពិន្ទុ)
លំហាត់ ៥. គណនាតម្លៃនៃផលបូក៖ $\sum_{k=1}^{5} (2k - 1)$ (៥ ពិន្ទុ)
លំហាត់ ៦. គណនាតម្លៃនៃផលបូក៖ $\sum_{i=1}^{4} i^2$ (៥ ពិន្ទុ)
លំហាត់ ៧. គេឱ្យស្វីតកំណត់ដោយ $u_1 = 1, u_2 = 3$ និងទំនាក់ទំនងលំដាប់២ $u_{n+2} = 3u_{n+1} - 2u_n$។ គណនា $u_3$។ (៥ ពិន្ទុ)
លំហាត់ ៨. សមីការលក្ខណៈនៃស្វីតលំដាប់ពីរ $u_{n+2} - 5u_{n+1} + 6u_n = 0$ គឺ៖ (៥ ពិន្ទុ)
លំហាត់ ៩. រកឫសនៃសមីការអ៊ិចស្ប៉ូណង់ស្យែល៖ $3^{2x-1} = 27$ (៥ ពិន្ទុ)
លំហាត់ ១០. ដោះស្រាយសមីការអ៊ិចស្ប៉ូណង់ស្យែល៖ $4^x - 3 \cdot 2^x + 2 = 0$ (៥ ពិន្ទុ)
លំហាត់ ១១. ដោះស្រាយវិសមីការអ៊ិចស្ប៉ូណង់ស្យែល៖ $2^{x+3} > 16$ (៥ ពិន្ទុ)
លំហាត់ ១២. ដោះស្រាយវិសមីការ៖ $\left(\frac{1}{2}\right)^x \le \frac{1}{8}$ (៥ ពិន្ទុ)
លំហាត់ ១៣. ដោះស្រាយសមីការលោការីត៖ $\log_2(x - 3) = 3$ (៥ ពិន្ទុ)
លំហាត់ ១៤. រកសំណុំចម្លើយនៃសមីការ៖ $\log_3 x + \log_3(x - 2) = 1$ (៥ ពិន្ទុ)
លំហាត់ ១៥. ដោះស្រាយវិសមីការលោការីត៖ $\log_5(x - 2) < 1$ (៥ ពិន្ទុ)
លំហាត់ ១៦. រកសំណុំចម្លើយនៃវិសមីការ៖ $\log_{\frac{1}{3}}(x) \ge -2$ (៥ ពិន្ទុ)
លំហាត់ ១៧. ដោះស្រាយវិសមីការ៖ $\log_2(2x - 4) > \log_2(x + 1)$ (៥ ពិន្ទុ)
លំហាត់ ១៨. រកគ្រប់តម្លៃ $x$ ដែលផ្ទៀងផ្ទាត់វិសមីការ៖ $\log(x) + \log(x - 3) \le 1$ (៥ ពិន្ទុ)
លំហាត់ ១៩. គណនាតម្លៃពិតប្រាកដនៃកន្សោម៖ $A = \sin\frac{\pi}{3} \cdot \cos\frac{\pi}{6} + \cos\frac{\pi}{3} \cdot \sin\frac{\pi}{6}$ (៥ ពិន្ទុ)
លំហាត់ ២០. គណនាតម្លៃលេខនៃកន្សោម៖ $B = 4\tan45^\circ - 2\sin30^\circ + \sqrt{3}\cot60^\circ$ (៥ ពិន្ទុ)
